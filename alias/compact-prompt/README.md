# compact-prompt

This is an **alias** for the [`compactprompt`](https://pypi.org/project/compactprompt/)
package. Installing it simply installs `compactprompt`.

```bash
pip install compact-prompt    # or: pip install compact_prompt
```

Both spellings resolve to this alias, which depends on `compactprompt`. Use the
library under its real import name:

```python
import compactprompt
```

See the [documentation](https://compact-prompt.readthedocs.io/) and the
[source repository](https://github.com/gtkcyber/compact_prompt).
